# 🛡️ Fake Job / Internship Scam Detector

A machine learning web app that detects fake job and internship postings using Random Forest + TF-IDF.

## Live Demo
👉 [Click here to try the app](#) ← replace with your Streamlit Cloud link

## Problem Statement
Thousands of students and freshers fall victim to fake internship/job scams every year — especially through WhatsApp, Telegram, and smaller job boards. This tool helps users instantly verify if a job posting is legitimate or a scam.

## Features
- Paste any job posting text — get instant risk score (0–100)
- Detects red flags: missing logo, unrealistic salary, scam keywords
- Shows green flags: detailed description, screening questions
- Safety tips based on verdict
- Model info and EDA insights

## Tech Stack
- **Python** — core language
- **Pandas, NumPy** — data analysis
- **Scikit-learn** — Random Forest, TF-IDF
- **imbalanced-learn** — SMOTE for class balancing
- **Streamlit** — web app framework
- **Matplotlib, Seaborn** — data visualization

## Dataset
[Real or Fake Job Postings — Kaggle (Shivam Bansal)](https://kaggle.com/datasets/shivamb/real-or-fake-fake-jobposting-prediction)
- 17,880 job postings
- 866 fake (4.8%), 17,014 real (95.2%)

## How to Run Locally
```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/fake-job-scam-detector

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```

## How to Retrain the Model
```bash
# Step 1 — Download dataset from Kaggle
# Step 2 — Run EDA
python phase1_eda.py

# Step 3 — Train model (generates scam_detector_model.pkl)
python phase2_model_training.py

# Step 4 — Launch app
streamlit run app.py
```
